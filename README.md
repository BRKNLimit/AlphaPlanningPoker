# 🚩 Team Alpha: Planning Poker
A high-performance, real-time estimation tool – Customized for Team Alpha.

## 🚀 Vision
Dieses Tool ist die interne Benchmark für effiziente Schätzrunden im Team Alpha. Fokus auf Schnelligkeit, Präzision und ein Design, das keine Kompromisse eingeht.

## ✨ Features
- **Team Alpha Branding:** Exklusives Schwarz/Rot-Design.
- **Instant Sessions:** Raum erstellen und Tickets schätzen ohne Overhead.
- **Frictionless Join:** Beitritt via Link oder ID.
- **Real-Time Voting:** Latenzfreie Synchronisation über Ktor WebSockets.
- **Zero-Persistence:** Alle Daten bleiben im RAM – maximale Sicherheit und Performance.

## 🛠 Tech Stack
- **Backend:** Ktor (Kotlin)
- **Engine:** WebSockets für Echtzeit-Kommunikation.
- **Frontend:** Responsive Web-UI im Team Alpha Corporate Design.
